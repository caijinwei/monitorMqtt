二、V-BOX安全须知
为避免不必要的财产损失，提高现场安全，在使用V-BOX之前，详细阅读本须知后，用户可以参照以下条款制定相应的V-BOX的安全使用管理制度：
1. 无关人员不得随意使用远程控制功能；
2. 对V-BOX的账户密码及机器设备序列号严格保密，避免泄露，如果密码被怀疑泄露，应立即重置密码；情况紧急的，可以致电维控的客服中心冻结账户的部分或一切功能；
3. 为了确保账户安全，用户应定期修改密码；
4. 在使用远程监控或者远程调试功能，请确认以下情况：
A. 网络畅通；
B. 人员安全，当设备处于远程调试状态时，远程操控人员通过电话等方式通知现场工作人员进行相关操作，以确保人身安全；
C. 设备及现场安全。
